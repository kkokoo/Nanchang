01

历史背景：风云变幻

大革命失败后的严峻时局

1924年至1927年，国共两党携手合作，掀起了轰轰烈烈的大革命浪潮。北伐战争势如破竹，工农运动蓬勃兴起，中国革命呈现前所未有的高涨局面。然而，正当革命即将取得重大胜利之际，国民党右派背叛革命，开始残酷镇压共产党人和革命群众。

1927年4月12日，蒋介石在上海发动反革命政变，大肆捕杀共产党员、工人领袖和革命群众，制造了震惊中外的“四一二惨案”。同年7月15日，汪精卫在武汉公开宣布“分共”，大规模逮捕屠杀共产党人，第一次国共合作彻底破裂，大革命宣告失败。

在短短几个月内，数十万革命志士惨遭杀害，中国共产党面临前所未有的生存危机。面对血雨腥风、白色恐怖，中国共产党人深刻认识到，只有拿起枪杆子，建立自己的武装，才能反抗国民党反动派的血腥屠杀，挽救中国革命。在这样生死存亡的历史关头，中共中央毅然决定：在南昌发动武装起义。

02

秘密准备：大旅社风云

南昌城内的秘密军事策划

1927年7月下旬，南昌城内暗流涌动，一场改变中国命运的武装起义正在秘密酝酿。中共中央派周恩来秘密抵达南昌，担任前敌委员会书记，全面领导起义筹备工作。起义指挥部秘密设在南昌市中心的江西大旅社，这里成为南昌起义的决策中枢。

7月27日，贺龙率领国民革命军第二十军、叶挺率领第十一军第二十四师，先后秘密进驻南昌城。这两支队伍都是中国共产党掌握或影响的精锐力量，成为起义的主力部队。朱德率领第三军军官教育团和南昌公安局警察武装，在城内配合行动，控制重要据点。

起义前夕，张国焘受共产国际指示赶到南昌，主张推迟甚至取消起义，认为敌我力量悬殊、起义必败。周恩来坚决反对，以辞职抗争，最终坚持原定8月1日凌晨起义计划。深夜，起义军将领在大旅社召开最后军事会议，部署详细作战任务，确定进攻路线、联络暗号、信号弹时间，为打响第一枪做好万无一失的准备。

7月25日：前敌委员会成立

周恩来抵达南昌，秘密成立起义指挥中枢。

7月27日：主力部队集结南昌

贺龙、叶挺部队秘密入城，起义兵力全部到位。

7月30日：起义决策之争

周恩来力排推迟起义意见，坚决执行原计划。

03

凌晨枪声：激战南昌城

震惊中外的武装战斗全过程

1927年8月1日凌晨2时整，三颗红色信号弹划破南昌漆黑的夜空，震惊中外的南昌起义正式打响。两万余名起义将士按照预定部署，从四面八方同时向南昌城内国民党守军发起猛烈进攻，枪声、炮声、冲锋号声响彻全城。

起义军兵分多路，重点攻克敌军核心据点。贺龙部主攻国民党江西省政府大楼，叶挺部歼灭敌军精锐部队，朱德部负责控制城北交通要道，防止敌军增援。战斗异常激烈，守军凭借坚固工事顽强抵抗，起义将士冒着枪林弹雨奋勇冲锋，展现出不怕牺牲、英勇无畏的革命精神。

经过四个多小时的浴血奋战，至清晨6时，南昌城内国民党守军被全部歼灭，共毙伤俘敌军3000余人，缴获大量武器装备。南昌城完全被起义军控制，鲜艳的红旗在南昌城头高高升起，中国共产党独立领导武装斗争、创建人民军队的伟大壮举宣告成功。

  • 战斗点 大旅社外围：起义军指挥部，率先肃清周边敌军。
  • 战斗点 匡庐中学：歼灭国民党第三军主力，激战四小时。
  • 战斗点 松柏巷天主堂：攻克敌军最后防线，胜利会师。
04

深远意义：建军的开端

改变中国革命走向的里程碑事件

南昌起义是中国革命史上具有里程碑意义的伟大事件，打响了武装反抗国民党反动派的第一枪，打破了大革命失败后中国革命的黑暗局面，标志着中国共产党独立领导革命战争、创建人民军队、武装夺取政权的正式开端。

南昌起义保存下来的革命火种，后来转战广东、福建、江西等地，历经艰难险阻，最终与毛泽东领导的秋收起义部队在井冈山胜利会师，共同创建中国第一个农村革命根据地，为中国革命开辟“农村包围城市、武装夺取政权”的正确道路奠定坚实基础。

1933年，中华苏维埃共和国临时中央政府正式将8月1日定为中国工农红军成立纪念日，即后来的中国人民解放军建军节。南昌起义铸就的“坚定信念、百折不挠、英勇无畏、为民奋斗”的八一精神，成为中国人民解放军的精神旗帜，激励一代代革命军人奋勇前进、为国为民。

建军节的由来

1933年，8月1日正式定为建军节，延续至今。

革命道路的转折

为井冈山会师、农村根据地建设奠定基础。

05

原始文献：历史的见证

留存至今的珍贵革命史料

南昌起义留存的原始文献，是见证这段伟大历史的珍贵实物，记录了起义决策、军事部署、革命主张等关键内容，为研究南昌起义提供最权威、最直接的一手史料，承载着革命先辈的初心与使命。

1927年8月1日，宋庆龄、毛泽东、周恩来、贺龙、叶挺、朱德、刘伯承等22位革命志士联名发表《中央委员宣言》，痛斥国民党反动派背叛革命、屠杀民众的滔天罪行，号召全国人民奋起反抗，建立革命政权，是南昌起义最重要的政治宣言。

此外，南昌起义前敌委员会会议记录、起义军作战命令、士兵誓词、革命标语、往来电报、新闻报道等原始史料，均真实还原起义全过程，见证中国共产党从血泊中站起来、拿起枪杆子、拯救中国革命的伟大转折，成为传承红色基因、弘扬革命精神的重要载体。
