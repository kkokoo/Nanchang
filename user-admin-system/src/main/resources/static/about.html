南昌历史云志

基于SpringBoot的地方历史文化数字化平台

2026年春季学期 · 软件工程课程成果

课题背景与使命

南昌作为英雄之城，承载着厚重的千年历史与鲜红的革命记忆。本应用旨在利用现代Web技术，实现南昌历史信息的数字化呈现、可视化交互及全天候传播。

我们的使命：让每一份历史档案动起来，让英雄的声音在数字时代继续共鸣。
